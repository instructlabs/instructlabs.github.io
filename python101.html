Introduction to Python

This is an introductory course on Python. The course teaches how to build Python programs groundup. The course is intended for someone with little or no experience programming in Python.

Modules

The learner is advised to visit the modules in the order given below.

1. Introduction

An introduction to Python language and about this course.

2. Numbers

Learn to do arithmetic computations in Python.

3. Strings

Learn to store and manipulate text in Python.

4. Lists

Learn to build and manipulate sequences of data.

5. Control Structures

Write programs that can run loops and make decisions on the fly.

6. List Comprehensions

Learn to build new lists derieved from existing ones.

7. Tuples

Tuples give a compact way to pack and unpack data.

8. Dictionaries

Dictionaries give a great way to search in a collection of items.

9. Sets

Learn to build sets and perform operations on them.

10. Functions

Python functions are a great way to build reusable components.

11. Modules and Packages

Build large programs spread across multiple files and directories.

12. Files

Learn to read and write files from Python.
